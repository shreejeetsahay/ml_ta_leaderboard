def check_submission_status(my_token):
    url = f"http://hadi.cs.virginia.edu:9000/submission-status/{my_token}"
    response = requests.get(url)

    if response.status_code != 200:
        print(f"❌ Error {response.status_code}: {response.text}")
        return

    attempts = response.json()
    for a in attempts:
        model_size = f"{a['model_size']:.4f}" if isinstance(a['model_size'], (float, int)) else "None "

        print(f"Attempt {a['attempt']}: Model size={model_size}, "
              f"Submitted at={a['submitted_at']}, Status={a['status']}")
    if attempts and attempts[-1]['status'].lower() == "broken file":
        print("⚠️ Your model on the server is broken!")

Model Architecture Requirements

Kindly ensure your autoencoder model adheres to the following specifications:
  • Input to Encoder: A full image tensor x of shape B × C × 256 × 256 (C=3 RGB).
  • Encoder (conv stack → bottleneck): Use convolutional blocks to downsample spatially; then flatten and apply a Linear layer to produce the latent code z of shape B × latent_dim.
  • Latent Dimension: latent_dim is a single integer representing the size of the bottleneck vector. Lower is better for compression.
  • Decoder (bottleneck → conv transpose stack): Map z through a Linear layer, reshape to a feature map, then use transposed convolutions / upsampling convs to reconstruct.
  • Output of Decoder: Reconstructed image ŷ of shape B × C × 256 × 256 with values in [0,1] (use Sigmoid at the end).
  • What to submit: A scripted model in the same spirit as the starter (Enc, Dec, wrapped by AE), saved via torch.jit.script(model) → model_ts.pt.

Submission Must-Haves

  • TorchScript artifact: Submit a .pt or .pth produced via torch.jit.script(...)
  • Input → Output shapes (RGB required):
    • Input: tensor [B, C, 256, 256] with C = 3 (RGB, values in [0,1]).
    • Latent vector: Shape B X latent_dim
    • Output: same shape as input [B, 3, 256, 256] (use a final activation so outputs are in [0,1]).
  • Deterministic eval: model.eval() should work
  • Artifact size limit: ≤ 23 MB for the TorchScript file.
